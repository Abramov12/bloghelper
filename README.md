# BlogHelper (bh)

CLI-инструмент-помощник для блогеров Instagram и TikTok.

Помогает быстро создавать черновики постов/Reels/TikTok: генерирует структуру текста, хуки, призывы к действию, хэштеги и сохраняет готовые шаблоны в Markdown.

## Зачем это нужно?

- Экономит время на подготовку описаний под разные платформы.
- Помогает соблюдать структуру поста (hook → контент → CTA).
- Упрощает работу с хэштегами и ключевыми словами.
- Всё работает локально в терминале — быстро и без лишних приложений.
